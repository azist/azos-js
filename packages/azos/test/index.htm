Test results are in the console (hit F12)

Azos uses ES6 modules. ES6 modules can not be loaded form local file system due to CORS limitations.
You need a server to serve those modules from local FS.
We use Parcel built-in dev server.

Execute the test by running this in command prompt, then open "localhost:1234" in browser 

  $ npm run test-bro
